Команды, доступные для ввода:
1. Exit - прекратить работу;
2. ПОКАЗАТЬ_МАГАЗИНЫ - Показывает магазины, которые уже есть в базе.
3. УДАЛИТЬ_МАГАЗИН.
Пример: УДАЛИТЬ_МАГАЗИНДевяточка
Сначала укажите название команды, затем название магазина. В одно слово, без пробелов.
4. ПОДКЛЮЧИТЬ_МАГАЗИН - перед работой с магазином необходимо выбрать требуемый.
Пример: ПОДКЛЮЧИТЬ_МАГАЗИНДевяточка
Сначала укажите название команды, затем название магазина. В одно слово, без пробелов.
5. Команда добавления магазина.
Пример: ДОБАВИТЬ_МАГАЗИНДевяточка
Сначала укажите название команды, затем название магазина. В одно слово, без пробелов.
6. Команда добавления товара.
Пример: ДОБАВИТЬ_ТОВАРВафли54
Сначала укажите название команды, затем название товара, в одно слово, без пробелов.
Затем укажите целое число — цену товара в рублях.
7. Команда добавления товара в магазин.
Пример: ВЫСТАВИТЬ_ТОВАРВафли
Сначала укажите название команды, затем название товара и магазина.
8. Команда получения информации о товарах во всех магазинах.
Пример: СТАТИСТИКА_ТОВАРОВ
Команда выводит для ассортимента:
общее количество наименований товаров,
среднюю цену товаров,
самый дорогой и самый дешевый товар,
количество товаров дешевле 100 рублей.
9. СТАТИСТИКА_МАГАЗИНА
перед работой с магазином необходимо выбрать требуемый с помощью команды. ПОДКЛЮЧИТЬ_МАГАЗИН.
После можно получить статистику. СТАТИСТИКА_МАГАЗИНА.
Сначала укажите название команды.
Команда выводит для ассортимента:
общее количество наименований товаров,
среднюю цену товаров,
самый дорогой и самый дешевый товар,
10. ПОЛУЧИТЬ_ТОВАР
Пример: ПОЛУЧИТЬ_ТОВАРВафли
Сначала укажите название команды, затем название товара.
Выводит информацию по товару. Из каталога товаров.
